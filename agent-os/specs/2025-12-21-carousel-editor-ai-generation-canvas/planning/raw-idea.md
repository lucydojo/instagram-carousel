# Raw Idea

**Feature Name:** Carousel Editor & AI Generation Canvas

**Description:**
A canvas-based Instagram carousel editor that combines AI-powered content generation with manual customization. The core experience is similar to tools like aiCarousels and PostNitro but with unique AI capabilities.

**Key Requirements:**
1. Canvas-style editor showing carousel slides side-by-side
2. AI content generation from prompts (text + images via nano banana pro)
3. "Optimization Options" panel for generation:
   - Social Platform targeting
   - Tone/Mood selection
   - Target Audience
   - Language
   - Saveable Presets
4. Style mimicking/inspiration from existing Instagram carousels
5. Per-slide editing (title, subtitle, paragraph, etc.)
6. Creator Info branding (name, handle, headshot - can be AI generated)
7. Color palette & typography customization
8. Natural language editing to modify generated content while keeping style consistent
9. Background effects, counters/corners options

**References:**
- Login page style: https://21st.dev/easemize/sign-in/default
- PostNitro-style optimization options
- aiCarousels-style canvas and creator info features

**Target Users:** Internal AI/Automation Agency team (demo platform)
